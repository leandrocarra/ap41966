Confira se o imóvel está pronto para receber a ligação:

Para seu imóvel receber a ligação, é necessário que ele tenha um padrão de entrada com livre acesso à caixa de medição para instalarmos o medidor de energia.

Para saber se seu imóvel já está preparado:
  • Consulte o síndico/administrador ou a construtora, para apartamentos ou salas comerciais.
  • Consulte um eletricista para realizar esse serviço, para comércios, industrias, residências e atividades rurais.
  • Benefíciários do cadastro único localizados na zona rural poderão solicitar a instalação gratuita do padrão de entrada caso forem elegíveis.
